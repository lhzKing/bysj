## 二、设计共识

### 2.1 风格基线（来自 DESIGN.md = Linear）

- **canvas**：`#fafafa`（light variant，从 Linear 的 `#010102` 反相）
- **surface ladder**：`#ffffff` → `#f4f4f5` → `#ebebec`（替代旧 surface-1/2/3）
- **hairline**：`#e6e6e8`（无阴影，靠 1px 描边分层）
- **ink ladder**：`#18181b` ink → `#3f3f46` muted → `#71717a` subtle → `#a1a1aa` tertiary
- **唯一 accent**：lavender `#5e6ad2`，仅出现在品牌标、focus ring、主 CTA、链路当前节点。**不允许出现 indigo `#6366f1` / 紫蓝旧 hover `#828fff` 之外的第二种 accent**。
- **语义色**：success `#27a644`、warn `#d97706`、error `#e5484d`，仅在 pill / dot 上使用，不染按钮主色。
- **dark 取景区**：摄像头预览专用 `#0f1011`，是整个 light 系统里唯一的暗面，承袭 Linear 原版气质。
- **字体**：Inter 400/500/600 + JetBrains Mono 400/500，display 用负字距（28px 用 -0.6px、36px 用 -1px、48px 用 -1.5px）。`shared/theme/tokens.js` 必须重写为这套体系。
- **圆角阶梯**：4 / 6 / 8 / 12 / 16，不允许 `rounded-[32px]` / `rounded-[48px]` / `rounded-[56px]` 这类巨型圆角再出现。
- **间距阶梯**：4 / 8 / 12 / 16 / 24 / 32 / 48；section 间距 ≤48，不再使用 96px 营销页节奏。

### 2.2 应用骨架（替代浮岛导航）

- **桌面（≥1024px）**：240px 固定侧栏 + 48px 顶栏 + 内容区。
- **平板（768–1023px）**：侧栏自动折叠为顶部汉堡，点击后从左滑出抽屉。顶栏保留面包屑与搜索。
- **手机（<768px）**：顶栏简化为汉堡 + 当前页标题 + 头像；侧栏抽屉占满高度；ScanHub 摄像头取景全屏。
- **不允许保留** `AppFloatingNav.vue` / `AppMobileNavDrawer.vue` / `AppContentFrame.vue`（浮岛胶囊 + mesh-bg + grid-accent 那一套）。

### 2.3 接口对接基线

- 所有页面真接后端，不允许 mock。`features/*/api/*.js` 已覆盖核心调用（见任务 F22 验证清单），重构时复用现有 axios 客户端 `core/api/request.js` 与各 feature 内的 api 模块。
- **新增**调用必须先读 `api-doc.md` 的对应章节，按 snake_case 响应、camelCase/snake_case 双向兼容请求的现状写。
- 所有错误码、状态码、字段名必须对齐 `api-doc.md`，**不允许在前端硬编码**（如把 `valid: false` 写成 `valid_chain: false`）。
- 必须验证 5 类用户路径：`SUPER_ADMIN` / `ADMIN` / `WAREHOUSE` / `LOGISTICS` / `PRODUCER`。详见 F22。

### 2.4 响应式验收基线

每个 UI 任务都必须满足：

- **≥1280px 桌面**：完整布局与预览页 1:1。
- **768–1023px 平板**：侧栏折叠为汉堡；多列网格降为单列或 2 列；表格保留水平滚动；Dialog 宽度 ≤80vw。
- **<768px 手机**：底部主操作粘性吸附；表格转卡片列表（每条记录一张卡）；Dialog 全屏；摄像头取景全屏覆盖；可通过 Chrome DevTools 的 iPhone 14 Pro / Pixel 7 模式逐页验收。

### 2.5 移除清单

任务流结束时，下列文件 / 类必须从代码库消失：

- 文件：`shared/components/layout/AppFloatingNav.vue`、`AppMobileNavDrawer.vue`、`AppContentFrame.vue`、`shared/components/prime/PrimeButton.vue`、`PrimeCard.vue`、`PrimeInput.vue`、`PrimeConfirmDialog.vue`、`PrimePromptDialog.vue`、`PrimeLoadingSkeleton.vue`、`PrimeToastHost.vue`。
- 全局 CSS 类：`.premium-card`、`.mesh-bg`、`.grid-accent`、`.nav-island`、`.hover-lift`。
- 字体加载：`Plus Jakarta Sans` 从 `style.css` 与 `index.html` 移除，统一走 Inter。
