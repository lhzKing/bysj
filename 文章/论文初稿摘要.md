# 摘要

本文面向工业企业配件供应链“来源不透明、流转难追踪、责任难界定”的现实痛点，设计并实现了一个基于 Spring Boot + Vue 的工业企业配件供应链溯源系统。系统采用前后端分离架构，后端以 Spring Boot、MyBatis-Plus、MySQL、Redis 为核心技术栈，前端采用 Vue 3 + Vite + ECharts，围绕“赋码—流转—查询—验链—统计”形成完整业务闭环。

在模型设计上，系统将配件规格、溯源快照与生命周期日志分层管理：以 trace_snapshot 保存当前状态，以 trace_lifecycle_log 保留完整事实链，兼顾高频查询与审计可追溯性。在安全方面，构建了 JWT 认证 + RBAC 授权 + Token 黑名单机制，并结合权限继承与细粒度权限控制，实现菜单级、接口级和业务动作级访问约束。为提升数据可信度，系统引入“Hash 链 + RSA 数字签名”双重防篡改机制，同时通过乐观锁重试策略保障并发扫描场景下的数据一致性。

在可视化与运维层面，系统提供了统计看板（KPI、趋势、地图、拓扑）与示例数据生成能力，支持教学演示与工程验证。项目实现结果表明，该系统能够有效支撑工业配件全生命周期追踪、异常定位与责任追溯，具备较好的可扩展性与工程落地价值，可为中小制造企业数字化供应链治理提供参考。

关键词：供应链溯源；Spring Boot；Vue 3；RBAC；数字签名；Hash 链

# Abstract

This thesis targets key pain points in industrial spare-part supply chains, including opaque origin information, poor circulation visibility, and difficult accountability. It designs and implements an industrial supply-chain traceability system based on Spring Boot and Vue. The system adopts a front-end/back-end separated architecture. The backend stack consists of Spring Boot, MyBatis-Plus, MySQL, and Redis, while the frontend uses Vue 3, Vite, and ECharts. It forms a complete closed loop covering code assignment, circulation recording, trace query, chain verification, and statistical analysis.

From the modeling perspective, the system separates part specification, trace snapshot, and lifecycle logs. The trace_snapshot table stores current state for high-frequency queries, while trace_lifecycle_log preserves the full factual chain for auditability. In security design, the system combines JWT authentication, RBAC authorization, and token blacklist management. With permission inheritance and fine-grained permission control, it enforces access constraints at menu, API, and business-action levels. To guarantee data trustworthiness, a dual anti-tampering mechanism based on hash chaining and RSA digital signatures is introduced. In addition, optimistic-lock retry is used to ensure consistency under concurrent scanning scenarios.

For visualization and operations, the platform provides dashboard modules (KPI, trend, map, and topology) and demo-data generation capabilities, supporting both teaching demonstrations and engineering validation. The implementation results indicate that the system can effectively support full-lifecycle traceability, anomaly localization, and accountability analysis for industrial spare parts, with good extensibility and practical deployment value for SME digital supply-chain governance.

Keywords: supply chain traceability; Spring Boot; Vue 3; RBAC; digital signature; hash chain
